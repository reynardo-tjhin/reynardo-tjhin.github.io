# About My Projects

This page shows a list of the projects that I actively worked on 🚀.

- [AI Server](/projects/AI%20Server/hardware)

Tags 🏷️: Building Computer | Large Language Models (LLMs) | llama.cpp

I first found out from a colleague that I could run Large Language Models (LLMs) at home. Soon after, I kept digging for more information about open source LLMs and got hooked. I started researching how to build an AI server. Most of what I learned came from [r/LocalLLaMA](https://www.reddit.com/r/LocalLLaMA/), where they discussed the latest open source models, build requirements, benchmarks, and a lot more. I also asked chatbots like Claude, ChatGPT, DeepSeek, etc. I picked up terms like "prompt processing", "token generation", and "memory bandwidth". This project aims to cover my AI server setup, the software/backend I use to power the open source LLMs and benchmarks of the recent models.

- [Media Server](/projects/Media%20Server/about)

Tags 🏷️: Software Development | Deployment

In this project, my aim is to learn software development from scratch to deployment to the real world. What motivated me was the need for a media server to store my personal files. Google Drive only allows 15 GB of data for free personal use 😢. On the other hand, cheap USB drives can be used for storage but viewing them requires a computer. Therefore, I plan to build a media server and deploy it to the internet for me and other users to store and retrieve their files on the web.
