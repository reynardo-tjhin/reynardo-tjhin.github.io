# About My Projects

This page shows a list of the projects that I work on actively 🚀.

- [AI Server](/projects/AI%20Server/hardware/)

Tags 🏷️: Building Computer | Large Language Models (LLMs) | llama.cpp

I first found out from a colleague that I could run Large Language Models (LLMs) at home. Soon after, I kept digging for more info about open source LLMs and got hooked. I started researching how to build an AI server. Most of what I learned came from [r/LocalLLaMA](https://www.reddit.com/r/LocalLLaMA/), where they discussed about the latest open source models, build requirements, benchmarks and a lot more. I also asked chatbots like Claude, ChatGPT, DeepSeek, etc. I picked up terms like "prompt processing", "token generation", and "memory bandwidth". The project aims to discuss my AI server setup, the software/backend I use to power the open source LLMs and benchmarks of the recent models.
